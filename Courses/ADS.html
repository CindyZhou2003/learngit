    Red-Black Trees Á∫¢ÈªëÊ†ë¶

    Âπ≥Ë°°ÁöÑ‰∫åÂèâÊêúÁ¥¢Ê†ëÔºåÁõÆÁöÑÊòØ‰ΩøÂä®ÊÄÅÊêúÁ¥¢Êõ¥Âø´
    ÂÆö‰πâÔºö
    1. Root property: The root is black.
    2. External property: Every leaf (Leaf is a NULL child of a node) is black in the Red-Black tree.
    3. Internal property: The children of a red node are black. Hence possible parent of red node is a black node.
    4. Depth property: All the leaves have the same black depth.
    5. Path property: Every simple path from root to descendant leaf node contains the same number of black nodes.
    The result of all these above-mentioned properties is that the Red-Black tree is roughly balanced.
    **Rules That Every Red-Black Tree Follows: **

    1. Every node has a color either red or black.
    2. The root of the tree is always black.
    3. Every leaf (e.i. NULL node) must be colored BLACK.
    4. There are no two adjacent red nodes (A red node cannot have a red parent or red child).
    5. Every path from a node (including root) to any of its descendants' NULL nodes has the same number of black nodes.
